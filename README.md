# Word Check
基于DFA算法以及Trie字典树开发的文本检测的服务

# Quick Start
```
docker-compose up -d
```
 - rpc run port: 5001
 - http run port: 5002

api doc: [swagger](http://localhost:5002/api-doc/index.html)
